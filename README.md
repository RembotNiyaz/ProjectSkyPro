Проект который в бэкенде будет готовить данные для отображения в новом виджете.

# 1. Декоратор логирования
## Описание
Модуль предоставляет декоратор log, который позволяет автоматически логировать выполнение функций. Декоратор может записывать логи как в файл, так и в консоль.

Установка
bash
# Установка необходимых зависимостей
pip install pytest
Использование
python
from decorators.log import log

# Пример использования с логированием в файл
@log(filename="mylog.txt")
def my_function(x, y):
    return x + y

# Пример использования с логированием в консоль
@log()
def another_function(x):
    return x * 2
Параметры декоратора
filename (опционально):

Если указан, логи записываются в указанный файл

Если не указан, логи выводятся в консоль

Форматы логирования
При успешном выполнении:

функция начало
функция ok
При возникновении ошибки:

функция error: тип_ошибки. Inputs: (аргументы), {именованные_аргументы}
Примеры использования
python
# Успешное выполнение
@log(filename="success.log")
def add(x, y):
    return x + y

# Обработка ошибки
@log(filename="error.log")
def faulty_func(x):
    return x / 'a'
# 2. Маскировка карты
