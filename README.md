# Описание
## Этот код реализует объектно‑ориентированную модель для работы с товарами (Product) и категориями товаров (Category). Подходит для базового учёта продукции в интернет‑магазине или складской системе.

# Основные классы
Product
Представляет отдельный товар.

# Атрибуты:

name — название товара (строка)
description — описание товара (строка)
price — цена (число с плавающей точкой)
quantity — количество на складе (целое число)
Дополнительные методы:

__str__() — строковое представление товара
__add__(other) — подсчет общей стоимости двух товаров одинакового типа
price — свойство с проверкой и возможностью снижения цены (с подтверждением)
quantity — свойство с проверкой на неотрицательное значение
Smartphone (наследник Product)
Дополнительные атрибуты:

efficiency — производительность
model — модель
memory — объем памяти
color — цвет
LawnGrass (наследник Product)
Дополнительные атрибуты:

country — страна-производитель
germination_period — срок прорастания
color — цвет
Category
Представляет категорию товаров.

# Атрибуты класса:

category_count — общее число созданных категорий
product_count — общее число товаров во всех категориях
Атрибуты экземпляра:

name — название категории
description — описание категории
products — список объектов Product в категории

# Дополнительные методы:

add_product(product) — добавление товара (с проверкой типа и дублирования)
remove_product(product_name) — удаление товара по названию
total_value() — подсчет общей стоимости всех товаров в категории
__str__() — строковое представление категории и её товаров
Новые возможности и улучшения
Добавлены методы для управления товарами в категории:
add_product() — добавление товара с проверками
remove_product() — удаление товара по названию
Подсчет общей стоимости товаров в категории:
total_value() — возвращает сумму стоимости всех товаров в категории
Обновлены счётчики:
category_count — увеличивается при создании новой категории
product_count — увеличивается при добавлении товара, уменьшается при удалении
Добавлена валидация входных данных:
Цена не может быть отрицательной
Количество — неотрицательное целое число
Обеспечена возможность динамически управлять списком товаров:
Можно добавлять и удалять товары из категории

# Пример использования

# Код демонстрирует:

Создание объектов Product (смартфоны, газонная трава).
Создание объектов Category (категории "Смартфоны", "Газонная трава").
Добавление и удаление товаров из категорий.
Подсчет общей стоимости товаров в категории.
Проверки счётчиков категорий и товаров.
Вывод информации о товарах и категориях. 

# Как запустить

Сохраните весь код в файл main.py.
Выполните команду:
CopyRun
python main.py
Ожидаемый вывод — информация о товарах, категориях и статистика счётчиков.
