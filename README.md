# Установка
## Клонируйте репозиторий:
CopyRun
git clone <ваш_репозиторий>
cd <папка_проекта>
## Создайте виртуальное окружение (опционально):
CopyRun
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
## Установите зависимости:
CopyRun
pip install -r requirements.txt
# Использование
## Запустите основной скрипт:

CopyRun
python main.py
Он выполнит все функции, сформирует отчеты и сохранит результаты в папку output/.

# Тестирование
## Для запуска тестов используйте pytest:

CopyRun
pytest tests/
Для мокирования функций в тестах используется pytest-mock.

# Основные функции
Загрузка транзакций из файла или источника
Формирование главной страницы и страницы событий
Расчет кешбэка по категориям
Инвестиционный план
Быстрый поиск по транзакциям
Генерация различных отчетов (по категориям, по дням недели, в рабочие и выходные дни)
Внесение изменений
Если вы хотите добавить новые функции или исправить существующие, редактируйте соответствующие модули в папке src/.
